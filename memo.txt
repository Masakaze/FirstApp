◾Heroku用の操作メモ
・herokuへのpush
 git push heroku master
・herokuでmigrate
 heroku run rake db:migrate
・herokuのrailsにアクセス
 heroku open

◾Gitの操作メモ
・ブランチの切り方
 git checkout -b ブランチ名
・ブランチの確認
 git branch
・masterへの戻り方
 git checkout master
・ブランチをマージ
 git merge ブランチ名
・ブランチの削除
 git branch -d ブランチ名
・GitHub自体へのコミット
 git push -u origin master

 ActiveRecord継承クラス.errors.full_msg
